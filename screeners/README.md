# Screeners

This directory contains Pine Script screeners for filtering and identifying trading opportunities.

## Overview

Screeners are specialized scripts designed to scan markets and filter stocks/assets based on specific technical criteria.

## Usage

1. Copy the Pine Script code from any `.pine` file
2. Open TradingView and go to the Pine Script Editor
3. Paste the code and click "Add to Chart"
4. Use TradingView's screener functionality to apply filters
